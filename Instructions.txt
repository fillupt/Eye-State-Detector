Recommended environment & setup

1) Install Python 3.12 (or compatible 3.11/3.12). Add it to PATH when installing if you want `py`/`python` available from the terminal.

2) Create a virtual environment (local to the project)

	Open PowerShell in the project root and run:

	py -3.12 -m venv eye-env

	# activate the venv (PowerShell)
